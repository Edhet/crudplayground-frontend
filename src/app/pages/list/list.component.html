CRUD Playground

What is this project?

This project is a CRUD playground with two different databases, Mongo and Postgres that can be switched in runtime.

Why was it made?

The idea behind the project was gaining experience in Springboot by building my own REST Api, learning JPA, Spring MVC, unit tests, software architecture in general and other things.

How it was made?

The project's backend was made in Springboot implementing a single REST endpoint for the two databases. The frontend was made in Angular with bootstrap
